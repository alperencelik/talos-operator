# Operator Manual

This is the operator manual for Talos Operator. It provides detailed information about how to use the operator, its features, and how to manage Talos Linux resources in a Kubernetes cluster. It tries to cover deeper topics and advanced usage of the operator. It's intended for users who want to understand the inner workings of the operator and how to leverage its capabilities effectively.
