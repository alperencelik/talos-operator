# Overview

This guide provides information on how to upgrade **talos-operator** to the newer versions, in case there are any breaking changes or important updates that need to be addressed during the upgrade process.

!!! warning **talos-operator** is under active development and might not follow the Custom Resource Definition (CRD) versioning principles. This means that **talos-operator** doesn't guarantee version updates on breaking changes and also there is no guarantee for the backward compatibility of the CRDs. Please be aware of this before using **talos-operator** in production environments.

!!! warning Before upgrading **talos-operator**, please make sure to backup your data and configurations.
